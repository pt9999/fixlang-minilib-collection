# Minilib.Collection.Deque

Defined in minilib-collection@0.5.1

Deque (double-ended queue).
Deque can be used as a FIFO such as a message queue.
When used as a FIFO, the amortized costs of `push_back()` and
`pop_front()` are `O(1)`.
